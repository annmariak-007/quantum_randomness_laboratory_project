
# Quantum Randomness Laboratory

A Python project developed as part of the **Project Q 30-Day Challenge**. This application explores quantum randomness by generating quantum random numbers, visualizing their distribution, performing statistical analysis, and comparing them with classical pseudo-random number generation.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/annmariak-007/quantum_randomness_laboratory_project/blob/main/quantum_randomness_laboratory.ipynb)

## Features

- 🎲 Generate Quantum Random Numbers (QRNG)
- 📊 Visualize distributions using histograms
- 📈 Perform statistical analysis of generated numbers
- ⚖️ Compare classical and quantum randomness
- 🐍 Built with Python and Matplotlib

## Technologies Used

- Python
- NumPy
- Matplotlib
- Qiskit / IBM Quantum (if applicable)

## Project Goal

The objective of this project is to understand the principles of quantum randomness and demonstrate how quantum-generated random numbers differ from classical pseudo-random numbers through visualization and statistical analysis.

## How to Run

You can run this project in either of the following ways:

* **Google Colab:** Click the **Open in Colab** badge at the top of this README to run the project directly in Google Colab.
* **Locally:** Clone the repository, install the required dependencies, and run `main.py` to execute the entire project.

> **Note:** While the project runs in Google Colab, the Qiskit histogram visualization may not render correctly due to Colab environment limitations. For the intended visualization experience, it is recommended to run the project locally.
